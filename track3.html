Track 3 — Microbus & objectif pro

Microbus, le preset qui structure tout

Microbus, c’est la formation qui a remis de l’ordre dans tout ce que j’avais accumulé : maths, anglais, logique, HTML/CSS/JS, Linux, Windows Server 2025… Ça a transformé mes réflexes en compétences plus carrées.

Maths Anglais Logique HTML / CSS JavaScript Linux Windows Server 2025 TFF en HTML/CSS/JS

Ce que Microbus m’a apporté

  • Bases solides Remise à niveau en maths, anglais et logique : plus à l’aise avec les exercices techniques, la doc et les échanges.
  • Web Pratique concrète de HTML/CSS/JavaScript : création de ce TFF comme une vraie mini-présentation d’artiste.
  • Systèmes Découverte et consolidation de Linux et Windows Server 2025 : gestion utilisateurs, partage, droits, logique d’admin.
  • Méthode Une façon plus structurée d’aborder les problèmes : analyser, tester, documenter, plutôt que “juste bidouiller”.

Preset : Objectif pro

Input Support IT
Output Admin systèmes & cloud
On-prem AD, GPO, WS 2025
Cloud Azure, M365
Sécurité MFA, identité, Zero Trust
Automatisation Scripts, Intune, déploiement postes

L’idée, c’est de garder le réflexe “pompier IT” du support, mais en prenant plus de hauteur sur l’infra complète : serveurs, cloud, sécurité.
